## Common Issues

### "Permission denied"
```bash
chmod +x sysmaint
sudo ./sysmaint
```

### "command not found: dialog"
```bash
# Ubuntu/Debian
sudo apt install dialog

# Fedora/RHEL
sudo dnf install dialog
```
