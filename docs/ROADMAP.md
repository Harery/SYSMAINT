# SYSMAINT Roadmap

**Version:** v1.0.0
**Last Updated:** 2025-12-27

---

## Current Version: v1.0.0 (Released 2025-12-27)

### Features

9 Linux distributions supported with core maintenance modules:
- Package management (update, upgrade, cleanup)
- System cleaning (logs, cache, temp files)
- Kernel management (old kernel removal)
- Service management (enable/disable services)
- User management (account cleanup)
- Security auditing (basic checks)
