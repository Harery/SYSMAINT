SYSMAINT V2.1.1 - QUICK REFERENCE GUIDE

HEADLINES (what it does)
| Area         | Behavior                                                             |
|--------------|----------------------------------------------------------------------|
| APT          | update, upgrade, autoremove, autoclean                               |
| Snap/Flatpak | Snap refresh; Flatpak update if installed                            |
| Firmware     | fwupd check if installed                                             |
| Cleanup      | DNS cache, journal (7d/500M), thumbnails, crash dumps, /tmp by age   |
| Diagnostics  | Failed services and zombie process checks                            |

COMMON COMMANDS
  sudo ./sysmaint                     # normal run (safe defaults)
  DRY_RUN=true JSON_SUMMARY=true ./sysmaint --dry-run --json-summary
  sudo ./sysmaint --auto --auto-reboot-delay 45 --json-summary
  sudo ./sysmaint --upgrade           # optional final full-upgrade
  ./sysmaint --help | less

KEY FLAGS
- Safety/flow: --dry-run, --json-summary, --auto, --auto-reboot-delay=N, --color=auto|always|never
- Logs: --log-max-size-mb=MB, --log-tail-keep-kb=KB
- Optional phases: --upgrade, --purge-kernels, --keep-kernels=N, --update-grub,
                  --orphan-purge, --fstrim, --drop-caches,
                  --browser-cache-report, --browser-cache-purge,
                  --security-audit
- Toggles: --no-snap, --no-flatpak, --no-firmware, --no-check-zombies, --no-clear-dns-cache,
           --no-journal-vacuum, --no-clear-crash, --no-clear-tmp
- Misc: --journal-days=N, --lock-wait-seconds=N, --progress=adaptive|dots|...,
        --flatpak-user-only | --flatpak-system-only

DEFAULTS (env/implicit)
  AUTO_REBOOT=true
  JSON_SUMMARY=false
  DRY_RUN=false
  COLOR_MODE=auto
  CHECK_ZOMBIES=true
  CLEAR_DNS_CACHE=true
  CLEAR_JOURNAL=true (7d/500M)  # change days via --journal-days=N
  CLEAR_THUMBNAILS=true
  CLEAR_CRASH_DUMPS=true
  CLEAR_TMP=true (safe by age)
  LOG_MAX_SIZE_MB=0 (disabled)
  KEEP_KERNELS=2 (only used with --purge-kernels)

JSON HIGHLIGHTS (v1.0)
  script_version, run_id, log_file, color_mode
  repos/kernel/system info, phase_timings, disk_usage, disk_deltas
  reboot_required, auto mode, progress metadata
  final_upgrade_enabled, upgraded_count, removed_count
  log_truncated, log_original_size_kb, log_final_size_kb
  zombie_count, zombie_processes (limited), security audit fields
  browser cache sizes and purge flag

TROUBLESHOOTING
  Exit 75 (lock): increase --lock-wait-seconds or use --force-unlock if stale
  Network hiccups: tune NETWORK_RETRY_* env vars
  No JSON file: set JSON_SUMMARY=true or pass --json-summary

PATHS
  Logs: /tmp/system-maintenance/sysmaint_*.log
  JSON: /tmp/system-maintenance/sysmaint_*.json
