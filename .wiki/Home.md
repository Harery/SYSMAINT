# SYSMAINT Wiki

**Automated System Maintenance Toolkit for Linux**

---

## Overview

SYSMAINT is a comprehensive system maintenance and automation toolkit designed for Linux environments. It provides automated package management, system cleanup, security auditing, and performance optimization across 9 major Linux distributions.

## Key Features

| Feature | Description |
|---------|-------------|
| **Package Management** | Automated updates for all supported distributions |
| **System Cleanup** | Remove unnecessary files and caches |
| **Security Auditing** | Scan for vulnerabilities and security issues |
| **JSON Telemetry** | Structured output for monitoring and logging |
| **Interactive TUI** | User-friendly terminal interface |
| **Docker Support** | Containerized deployment option |

---

## Supported Distributions

- Ubuntu (22.04, 24.04)
- Debian (12, 13)
- Fedora 41
- RHEL 10
- Rocky Linux 9
- AlmaLinux 9
- CentOS Stream 9
- Arch Linux
- openSUSE Tumbleweed

---

## Quick Start

```bash
git clone https://github.com/Harery/SYSMAINT.git
cd SYSMAINT
chmod +x sysmaint
sudo ./sysmaint --dry-run
```
