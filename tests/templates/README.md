# Test Suite Templates

This directory contains templates for creating new test suites.

## Available Templates

1. **basic_suite.sh** - Basic test suite template
2. **os_family_suite.sh** - OS-specific test suite template
3. **feature_suite.sh** - Feature-specific test suite template

## Usage

Copy a template and modify for your needs:

```bash
cp templates/basic_suite.sh tests/test_suite_my_feature.sh
# Edit tests/test_suite_my_feature.sh
chmod +x tests/test_suite_my_feature.sh
```
