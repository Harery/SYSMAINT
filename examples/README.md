# Example Configurations

This directory contains example configurations for common use cases.

## Files

| File | Description |
|------|-------------|
| `minimal.yaml` | Minimal configuration |
| `production.yaml` | Production server configuration |
| `development.yaml` | Development workstation |
| `compliance-hipaa.yaml` | HIPAA compliance |
| `compliance-soc2.yaml` | SOC 2 compliance |
| `fleet.yaml` | Fleet management |

## Usage

```bash
cp examples/production.yaml ~/.config/pulse/config.yaml
```
